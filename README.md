# CRM Pakora

CRM/Torre de control COD para operación Dropi + Shopify — Colombia y México

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
- Vercel

## Getting Started

Use Node.js `v24.18.0` from `.nvmrc`.

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the placeholder home page.

For a production build:

```bash
pnpm build
```
